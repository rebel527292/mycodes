Digital Marketing Agency Landing Page
A clean, responsive, and professional landing page template designed for Digital Marketing Agencies. This project is built using semantic HTML5 and modern CSS3 with a focus on high conversion and fast load times.

🚀 Features
Responsive Design: Fully optimized for mobile, tablet, and desktop views using CSS Flexbox.

Hero Section: A bold call-to-action (CTA) to drive lead generation.

Services Grid: Clean, card-based layout to showcase core offerings.

Lead Capture: A styled contact form for potential client inquiries.

Modern Aesthetics: Professional blue-and-white color palette with minimal typography.

🛠️ Built With
HTML5: Semantic structure for better SEO and accessibility.

CSS3: Custom styling, Flexbox layouts, and box-shadow effects.

No Frameworks: Lightweight and dependency-free for maximum performance.
